Wat ik hebt geleerd

Tijdens deze periode heb ik bij het keuzevak HTML/CSS heel veel geleerd. Voordat ik het keuzevak begon koos ik HTML/CSS omdat ik ge√Ønteresseerd was in de taal. Daarnaast leek het me handig om zelf te weten hoe je sites bouwt zodat ik hier zelf mee aan de slag kan en zodat ik website bouwers beter kan begrijpen en hiervoor realistische ontwerpen maken.

Ik heb ooit al eerder met HTML gewerkt maar dat was lang geleden. Aan het begin van het keuzevak leerde ik weer kennis maken met de basics van HTML. Daarbij leerde ik om de programma's Visual Studio Code en GitHub te gebruiken. Hier had ik van tevoren nog nooit van gehoord. Naast de basics van HTML leerden we over CSS, waar ik nog nooit mee had gewerkt. We leerden eerst wat CSS inhield en hoe je dit kon koppelen. Daarna heb ik veel kunnen leren over positionering, opmaak en vooral layout met HTML en CSS. Dit heb ik door middel van de lesuitleg en de opdrachten tijdens het keuzevak kunnen leren.

Toekomstige actiepunten

- Consistent zijn met opmaak via HTML of via CSS schrijven per document.
- Consistent zijn met afmetingen in % of px zetten per document.
- Meer met classes werken.
- Pagina's schalend maken
- Code duidelijker indelen (voor overzicht)

Wat ik zou willen maken

- Ik zou graag sites willen maken met klikbare elementen, die elementen laten veranderen en klikbare elementen die je naar andere pagina's leiden.

- Verder zou ik sites willen maken met sidebars en willen werken met (achtergrond)video's.

- Ook wil ik mijn eigen portfolio maken met al mijn toekomstige projecten etc.
